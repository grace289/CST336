What is a Virus?

What is a virus? Virus Types Virus Symptoms Prevention and Removal



A computer virus is a malicious software program also called malware. This "virus" can infect a computer by modifying or deleting data files, or cause a software program to work in an unexpected way. A virus lives on a host computer and can replicate itself when executed. Viruses can steal user data, delete or modify files & documents, record keystrokes & web sessions of a user. Viruses can also steal or damage hard disk space, it can slowdown a computers processor as well.
